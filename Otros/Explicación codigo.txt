3. Creación de la Tabla (init_db)
conn = sqlite3.connect(...): Abre una conexión con el archivo .db. Si no existe, lo crea.

conn.commit(): Guarda los cambios permanentemente.
conn.close(): Cierra la conexión para liberar memoria.

fecha_actual = datetime.now().strftime(...): Obtiene la hora de la computadora y le da un formato legible (día/mes/año, hora:minutos).
try... except... finally:: Es una estructura de seguridad. Intenta hacer algo y, si hay un error, lo captura para que el programa no se detenga bruscamente.
cursor.execute('INSERT INTO ... VALUES (?, ?, ?, ?)', (...)): Inserta los datos del usuario en la tabla. Los signos de ? son por seguridad (evitan ataques de inyección SQL).
finally: conn.close(): Pase lo que pase (error o éxito), se asegura de cerrar la conexión a la base de datos.

___________________________________________________________________________________________________________________

 1. db.py (La Base)
  Este es el archivo inicial desde el punto de vista de la infraestructura. Se encarga de crear la base de datos SQLite
  (sentiment_iq.db) y la tabla reviews. Sin este archivo, no hay dónde guardar la información que los demás procesarán.
   * Función: Gestión de persistencia (SQL).

  2. textprocessor.py (El Utilitario)
  Este archivo contiene la lógica de limpieza de texto. Es una pieza fundamental que transforma el lenguaje humano
  desordenado en "tokens" (palabras clave) limpios.
   * Función: Limpieza de datos (Stopwords, tokenización, expresiones regulares).

  3. sentiment_model.py (La Inteligencia)
  Aquí reside el motor de Inteligencia Artificial. Utiliza textprocessor.py para limpiar los datos y luego emplea la
  librería scikit-learn para entrenar un modelo que aprenda a distinguir entre comentarios positivos y negativos.
   * Función: Machine Learning (Entrenamiento y Predicción).

  4. main.py (El Orquestador)
  Este es el archivo principal que debes ejecutar para que todo el sistema funcione en conjunto. Importa a los otros
  tres y ejecuta un "pipeline": busca reseñas pendientes en la base de datos, las pasa por la IA y genera un reporte
  estratégico final.
   * Función: Integración y ejecución del flujo de trabajo completo.
